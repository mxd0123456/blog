[学习]C++逐行打印1~100的数

Sunday, July 17th 2022, 9:19:40 AM

一道简单的练习题

说明：
逐行打印1~100的数

需要输入的数据：
无

需要输出的数据：
1～100所有的数字，一行一个整数。

代码：

#include
using namespace std;
int main(){
int a; //定义整型变量a，作为输出内容
for(a=1;a<=100;a++){ //for循环，循环一百次
cout<<a<<endl;
}
return 0;
}
