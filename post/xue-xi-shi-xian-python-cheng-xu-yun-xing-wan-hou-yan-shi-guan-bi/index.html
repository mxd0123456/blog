实现Python程序运行完后延时关闭

这是一篇帮助初学者的教程

假如说我这个程序是计算xxx的，但计算完毕得到结果就自动关闭了，让人看的时间都没有

这时我们需要一个参数

调用时间模块里的sleep函数

sleep，顾名思义，睡觉、延时

#单位为秒，可以设置为0.4，那就是400毫秒
time.sleep(2)
注意，一定要先
实现Python程序运行完后延时关闭

import time
#一定要导入模块
现在我们看一下具体场景
比如我要Print123，我想让他输出后2秒自动关闭，那代码就是：
——————————————————————————————

-- coding:UTF-8 --

import time
#先导入time模块

print("123")
#输出123

time.sleep(2)
#两秒关闭
