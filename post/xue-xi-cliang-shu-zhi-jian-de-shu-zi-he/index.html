[学习]C++两数之间的数字和

Sunday, July 17th 2022, 9:24:49 AM

十分简单！

简要说明：
从键盘输入两数，统计两数间的整数和。

需要输入的数据：
一行，两个整数a和b，测试数据保证a<b。

需要输出的数据：
一个整数，a到b的所有数字之和。

代码：

#include
int main()
{
using namespace std;
int a,b; //定义整型变量a，b
cin >> a >> b; //cin 输入
double sum=((double)(a+b)/2*(b-a+1)); //计算（首项+末项）x 项数 ÷ 2
//有一个（double），是强制类型转换
cout<<sum<<endl; //cout 输出
return 0;
}
