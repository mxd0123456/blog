在LinuxMint上安装AWESOME桌面环境

Monday, July 11th 2022, 11:26:41 AM

AWESOME是一个很简洁的桌面环境

可转载，但请标上原文地址及作者！

其实很简单，只要执行几条命令就行了

基于Ubuntu的发行版都可用！

记得一定要apt-get update一下！！！

命令：

sudo su
#root权限
apt-get update
#更新软件
apt-get install awesome
#安装AWESOME
