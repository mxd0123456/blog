Linux Mint登录到桌面时黑屏，显示一个对话框“cinnamon-session-cinnamon not-found”

Sunday, July 10th 2022, 9:42:04 PM

就像这样，我猜测应该是安装了unity桌面环境时某个软件包与cinnamon的某个软件包起了冲突

其实解决办法很简单：
1.点击okay
2.使用组合键 Ctrl + Alt + F1 进入指令界面
3.输入帐号密码（有数字记得点Num Lock）
4.执行：
sudo apt-get install cinnamon-screensaver mint-meta-cinnamon nemo nemo-fileroller nemo-share
5.输入密码
6.执行：
sudo reboot
#第4步是修复软件包
#第6步是重启
